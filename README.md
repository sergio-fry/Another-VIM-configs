# Another-VIM-configs
My VIM configs

## Install

```bash
$ git clone https://github.com/sergio-fry/Another-VIM-configs.git ~/.vim
$ ln -s $HOME/.vim/vimrc ~/.vimrc
$ cd ~/.vim
$ curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
    https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
$ vim +PlugInstall
```
